Milestone: Pre-implementation Tollgate

Check if the project is on the right track to deliver the agreed benefits / goals

Make the decision on whether to reject the project, loop to redo elements of defintion (agreed at the meeting), or proceed to next stage

Project Controls
  • All project controls were appropriately applied and necessary sign-off taken
Business Readiness
  • The business is ready to start the implementation.
  • All stakeholders (including sponsor) are still in place and actively engaged.  There is full acceptance of the change and commitment to implement and embed the change.
  • Business leaders, managers and employees are aware of the change.
  • Any awareness building or notification activities have taken place with customers.
  • Any helpdesk or other support channels have been setup or briefed.
  • All communication material to support implementation is signed off and ready to be distributed.
  • The implementation site schedule is fully detailed and go live / implementation dates have been communicated.
  • Site specific activity checklists have been defined, with responsibilities and dates.
  • Reporting route and frequency for activity completion has been confirmed.
  • Implementation management team confirmed.  Delivery partner points of contact are confirmed.
  • Implementation issue categories and escalation routes have been confirmed.
  • Implementation management calls are scheduled on a regular basis.
  • All implementation support material is signed off and ready to be distributed.
  • All training material to support implementation is signed off and ready to be distributed.
  • If applicable, the training schedule is confirmed and coaches/trainers/facilities/participant time has been agreed and booked.  Communications to support training delivery are ready.
  • If applicable, organisation alignment and operations alignment changes are scheduled or have been completed (e.g. process changes, policy changes, job description changes etc.).
  • Tracking and reporting of the change and benefits is active.
Solution Readiness
  • The change solution is ready for implementation.
  • The change solution delivers the change vision/objectives and fully meets the business/detailed requirements.  Traceability has been reviewed.
  • The change solution is free of defects following testing or review.
  • Employee or customer feedback from testing or review has been acted on or factored into change activity.
  • Support Channels changes are fully ready to be implemented.
  • System changes are fully ready to be implemented.
  • Data changes are fully ready to be implemented.
  • Customer-related changes are fully ready to be implemented.
  • Channels-related changes are fully ready to be implemented.
  • Product and proposition related changes are fully ready to be implemented.
  • Operations changes (e.g. process, procedures, service management, operations management etc.)  are fully ready to be implemented.
  • Supplier changes are fully ready to be implemented.
  • Facilities changes are fully ready to be implemented.
  • People and Organisation changes are fully ready to be implemented.
  • Governance changes are fully ready to be implemented.
  • Management Information changes are fully ready to be implemented.
