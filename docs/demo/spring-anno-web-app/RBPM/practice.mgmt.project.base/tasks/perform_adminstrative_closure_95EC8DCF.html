Task: Perform Adminstrative Closure

Administrative Closure is required after the programme, project, or phase has achieved its objectives or been terminated for other reasons. It consists of verifying and documenting project or programme results to formalise acceptance of project or programme deliverables by the Project Sponsor. It includes collection of project records, ensuring specifications are up-to-date, analysis of project or programme success and effectiveness and archiving such information for future reference.

You must close each project or programme phase properly to ensure that information is not lost.
