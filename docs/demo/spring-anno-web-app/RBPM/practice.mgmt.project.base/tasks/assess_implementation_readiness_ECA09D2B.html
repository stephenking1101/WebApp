Task: Assess Implementation Readiness

Purpose

Identify open issues pending resolution prior to implementation and ensure business readiness.

Procedure

  1. Identify open issues pending resolution including identification number and a brief description.
  2. Summarise the number of business as usual (BAU) to be resolved after implementation. Only low priority defects should be included in this section. Medium / High / Critical should be resolved or identified in the open issues section
  3. Certify the project is ready for BAU including updates to user manuals, training is in place, user access is granted
  4. Identify all business process preparations that should be complete prior to implementation and the date completed. These can include new or updated procedures, scripts, forms, user access, site preparations or other deliverable that supports Business Operations. Any exception to the full completion of any documented requirement and its potential impact to a successful implementation should be noted in the Exceptions column and agreed to by all approving stakeholders.
  5. Identify all required documents that have been completed, approved and issued to all relevant parties. Any exception to fully complete, approved and distributed documents should be noted in the Exceptions column and agreed to by all approving stakeholders. The documents listed in the table are for example and represent potential document requirements; any listed documents not required by the project may be deleted. This can serve as a checklist for audit.
  6. Obtain implementation authorisation sign off which should include all relevant project team members, for example, project sponsor, project manager(s), BAU support team, and so on.

If your project was set to deliver on sustainability goals, assess if the project is on track to deliver them.

Also, if you project doesn't have any defined impacts on sustainability goals, you can still refer to Considerations for assessing impact to check if anything can be included during the deployement.
