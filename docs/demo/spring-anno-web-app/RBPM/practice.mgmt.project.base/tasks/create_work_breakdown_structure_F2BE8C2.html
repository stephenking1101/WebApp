Task: Create Work Breakdown Structure

Purpose

A work breakdown structure is a hierarchically structured groupings of project elelments that organises and defines the total scope of the project. Each descending level is an increasingly detailed definition of a project component.

Work breakdown structure is the basis for further project planning and will help in estimating the duration of the project, determine the required resources and work schedule.

Procedure

  1. Identify the highest level components of work to be accomplished
  2. Break work process into small components
  3. Name the component
  4. Make sure work has been broken down to the lowest level desired
