Task: Plan Deployment / Implementation

Deployment / implementation planning requires a high degree of customer collaboration and preparation.

Procedure

  1. Specify what will be deployed, to whom, when, and the roles and responsibilities of those involved in deployment.
  2. Identify Testing requirements. See, Global Test Framework.
  3. Identify User Support requirements. See, User Documentation guidelines.
  4. Identify User Training requirements. See, User Training guidelines.
